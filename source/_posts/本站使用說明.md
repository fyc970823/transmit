由於不知道第一篇文要寫什麼，所以來寫如何使用本網站，讓你可以順利地參觀！

# 首頁
## 文章
文章在[首頁](https://fyc970823.github.io/transmit/)滾輪鍵往下滑就看的到了唷，如果文章太多在最下方可以切換頁數。（但近期應該不會有這個困擾）

## 設定（齒輪）
齒輪要在[首頁](https://fyc970823.github.io/transmit/)往下滑一點才會出現（其他頁面會直接顯示），按下後可以調整亮/暗模式、字體大小、還有一鍵滑到底等功能。

## 下滑百分比
一樣要在[首頁](https://fyc970823.github.io/transmit/)往下滑一點才會出現，數字越小表示離文章頂越靠近，數字越大表示離文章底越靠近。

# 歸檔
所有文章都會出現在這邊，發布時間、標題清晰明瞭，要找文章除了可以在[首頁](https://fyc970823.github.io/transmit/)找之外，在[歸檔](https://fyc970823.github.io/transmit/archives/)找也是不錯的選擇！

# 分類
這裡跟前面的[首頁](https://fyc970823.github.io/transmit/)和[歸檔](https://fyc970823.github.io/transmit/archives/)不同，在[分類](https://fyc970823.github.io/transmit/categories/)的文章都是經過分類的，若不知道主題名稱但知道文章內容的主題方向，在這裡找或許會方便許多。

# 標籤
在[這裡](https://fyc970823.github.io/transmit/tags/)，是關於我的所有標籤，只要出現在上面的標籤，基本上我都略懂一二，我超強哈哈。

# 關於
這裡有分兩大類，分別是[關於我](https://fyc970823.github.io/transmit/about/)和我的[Github](https://github.com/fyc970823)。

## 關於我
這裡面有許多關於我的訊息，有我的自我介紹、興趣、獲得的競賽、檢定與獎項、聯絡方式與座右銘。~~不瞞你說座右銘是ChatGPT生的~~

## Github
這裡面有我網站的專案，有興趣可以看看。

# 友站
這裡面有很多我的好朋友們，點進去可以連接到朋友們的個人檔案（IG,Github pages等）。

## 加入友站列說明
若想出現在本站，在加入本站後，在下面留言，格式如下：
``` bash
  - name: #你的名稱
    link: #你的傳送連結
    description: #簡介
    avatar: #頭像連結
```
我就會在某天~~等我注意到的時候~~把你加上友站列表當中。
